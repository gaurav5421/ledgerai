# Failure Catalog

This document catalogs failure modes encountered during development and how LedgerAI handles them. Honest documentation of failures builds more credibility than any success story.

## Format

Each entry includes:
- **What happened** — the failure as observed
- **Root cause** — why it happened
- **How the agent handles it** — the mitigation or fix
- **Lesson** — the generalizable takeaway

---

*Entries will be added as the system is built and tested.*
